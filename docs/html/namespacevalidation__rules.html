◆ validate_phone()

bool validation_rules.validate_phone ( str phone)

Walidacja polskiego numeru telefonu.

Sprawdza, czy podany ciąg znaków jest prawidłowym numerem telefonu. Obsługuje różne formaty zapisu (z prefiksem +48, bez prefiksu, z myślnikami, spacjami i nawiasami), ale ostatecznie numer musi składać się z dokładnie 9 cyfr.

Parameters
phoneCiąg znaków reprezentujący numer telefonu do sprawdzenia
Returns
True jeśli numer telefonu jest prawidłowy, False w przeciwnym wypadku
