Moduł zawierający funkcje walidacyjne dla danych wprowadzanych przez użytkowników.

Zapewnia zestaw funkcji do weryfikacji poprawności danych, takich jak NIP, numery telefonów, adresy email, kody pocztowe i inne dane adresowe.
