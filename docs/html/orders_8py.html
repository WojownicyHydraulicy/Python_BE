Detailed Description

Moduł obsługujący routing dla operacji związanych z zamówieniami.

Zapewnia endpointy API do tworzenia zamówień, pobierania listy zamówień, aktualizacji zamówień i zarządzania ich statusem. Moduł implementuje logikę biznesową potrzebną do obsługi całego cyklu życia zamówienia w systemie.
